Aapda-Predict — GeoAI Disaster Command Center

A full-stack GeoAI web platform fusing Sentinel-1 satellite radar with computer vision to map flood extents through 100% cloud cover — solving the "optical blindness" problem for disaster response teams.

Problem & Solution
Problem

Optical satellite imagery is blind during storms — exactly when flood-response teams need visibility most.

Solution

Radar-based (SAR) imaging that sees through cloud cover, paired with CV-based noise filtering for real-time water detection.

Key Features
Live data-processing pipeline filtering radar speckle noise via morphological operations
98% signal accuracy achieved for real-time water detection
Ranked 14th nationally (Top 20) out of 200+ competing teams at SkyHack 2025
Tech Stack
PythonStreamlitOpenCVSentinel-1 SARAltairNumPy
Challenge & What I'd Improve

Radar speckle noise made naive water-detection thresholds unreliable — tuning the morphological filtering to hit 98% accuracy under a hackathon clock was the hardest part. Next step would be swapping the rule-based filter for a trained segmentation model.
